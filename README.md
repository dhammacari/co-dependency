# ☸️ Madhyamaka: The Two Truths
### "Between Appearance and Emptiness"

A minimalist, interactive web project exploring the core tenets of **Madhyamaka (Middle Way) Philosophy** by Acharya Nagarjuna. This project serves as a digital "Dharma Lamp," illustrating the interplay between **Conventional Truth** (Samvrti-satya) and **Ultimate Truth** (Paramartha-satya).

---

## 🕯️ Philosophy in Motion
In this project, code becomes a medium for contemplation:
* **Conventional Truth:** The visible text and interface—tools for communication in the relative world.
* **Ultimate Truth:** The dissolution of form upon interaction—representing **Sunyata** (Emptiness), the lack of inherent existence.

> *"Whatever is dependently co-arisen, that is stated to be emptiness."* — Nagarjuna

---

## ✨ Features
* **Minimalist Artisan Design:** Deep dark mode aesthetics for focused contemplation.
* **Bilingual Wisdom:** Toggle between Thai and English to reach a global audience.
* **Interactive Sunyata:** Experience the "emptiness" of language through a blur-and-dissolve hover effect.
* **Pure Logic:** Built with clean HTML, CSS, and Vanilla JavaScript. No unnecessary dependencies.

---

## 🛠️ Deployment
This project is designed to be "Live" instantly.
1. Clone the repository.
2. Open `index.html` in any browser.
3. Contemplate the space between the words.
